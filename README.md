# 🛍️ Cartify Fashion Hub

A modern and responsive **React.js Fashion E-Commerce Website** where users can browse trendy fashion products, add items to their cart, and enjoy a smooth shopping experience.

---

## 🚀 Features

* 🏠 Responsive Home Page
* 👕 Fashion Products Display
* 🔍 Product Search
* 🛒 Add to Cart Functionality
* ➕ Increase/Decrease Quantity
* ❌ Remove Items from Cart
* 💰 Automatic Total Price Calculation
* 📱 Mobile-Friendly Design
* ⚡ Fast and Interactive UI using React

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Bootstrap (if used)
* React Router DOM (if used)

## ⚙️ Installation

1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```

## 🎯 Project Objective

The main objective of **Cartify Fashion Hub** is to provide a user-friendly and responsive online fashion shopping experience while demonstrating modern frontend development using React.js.
